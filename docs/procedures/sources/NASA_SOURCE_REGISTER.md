# ARES-1 NASA Source Register for Emergency Procedure Manuals

This register supports the six ARES-1 simulated mission emergency procedures. NASA sources are used as technical standards or references; ARES-specific scenario values, action effects and thresholds remain separately classified.

## Source policy

- **NASA_STANDARD:** A requirement from an identified NASA standard and requirement ID.
- **NASA_REFERENCE:** NASA handbook, technical report, paper, training material or architecture reference.
- **DERIVED_PHYSICS:** Calculated by the frozen simulator from conservation laws or equations.
- **ARES_ASSUMPTION:** Project-specific operational sequencing, physiology mapping or simplification.
- **ARES_RELEASE_CONFIGURATION:** Values and capabilities in the current release scenario and strict executable contract.

## Non-claim rule

These sources do not make ARES-1 an official NASA system. The manuals must never be described as NASA-approved, flight-certified, validated by NASA, or suitable for real spacecraft operations.
